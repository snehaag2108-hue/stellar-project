# 🔥 Stellar Token Burn Mechanism

## 📌 Project Description

This project implements a **Token Burn Mechanism** using the **Stellar Soroban smart contract platform**. The goal is to create a controlled way to reduce the total supply of a token over time, increasing scarcity and potentially enhancing token value.

The contract enables users or authorized entities to permanently remove tokens from circulation in a transparent and verifiable manner.

---

## 🚀 What It Does

* Allows token holders or authorized accounts to **burn (destroy) tokens**
* Reduces the **total token supply permanently**
* Ensures all burn transactions are **recorded on-chain**
* Provides a **secure and decentralized mechanism** for supply control

---

## ✨ Features

* 🔥 **Token Burning Functionality**
  Users can burn tokens, removing them permanently from circulation.

* 🔒 **Access Control (Optional)**
  Restrict burn functionality to specific accounts (e.g., admin or owner).

* 📉 **Supply Reduction Tracking**
  Automatically updates and reflects reduced total supply.

* ⚡ **Efficient Smart Contract (Soroban)**
  Built using Stellar’s Soroban framework for speed and low cost.

* 🧾 **Transparent Transactions**
  All burns are recorded on the Stellar ledger for full transparency.

---

## 🛠️ Tech Stack

* **Blockchain:** Stellar
* **Smart Contracts:** Soroban (Rust)
* **Language:** Rust
* **Tools:** Stellar CLI, Soroban SDK
